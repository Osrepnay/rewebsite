This is an interactive demonstration of link-state routing.

Concept

Controls

Click on a node to select it. Hold Control while selecting a node to drag it. Press G to cycle through the editing modes.

Building Mode: Click an empty spot to create a node. Click another node to add a connection, or to delete the existing connection.

Testing Mode: Select two nodes to send messages between them.

Viewing Mode: Select any node to view its routing table.
